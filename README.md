# Sudoku Solver

A minimal web-based Sudoku application with auto-solve functionality.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser to: http://127.0.0.1:5000

## Features

- Interactive 9x9 Sudoku grid
- Auto-solve using backtracking algorithm
- Load sample puzzle
- Clear grid
- Visual distinction between user input and solved cells

## Usage

- Enter numbers (1-9) in the grid cells
- Click "Load Sample" to try a pre-loaded puzzle
- Click "Auto Solve" to solve the current puzzle
- Click "Clear" to reset the grid
