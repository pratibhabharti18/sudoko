# Project Structure

## Directory Organization

```
.
├── .kiro/              # Kiro AI assistant configuration
│   ├── steering/       # AI guidance documents
│   └── specs/          # Feature specifications (created as needed)
└── .vscode/            # VSCode settings
```

## Conventions

### Kiro Configuration

- **Steering files**: Located in `.kiro/steering/` - provide context and guidelines for AI assistance
- **Spec files**: Located in `.kiro/specs/{feature-name}/` - contain requirements, design, and tasks for features

### Code Organization

To be defined as the project structure develops.

## File Naming

- Use kebab-case for directories and feature names
- Follow language-specific conventions for source files
